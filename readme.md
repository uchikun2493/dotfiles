## docker

pyhton環境を整えるためのファイル

```
$ # cd setup_ubuntu or setup_python
$ # build
$ sh setup.sh
$ # sample run
$ sh python_run.sh
```

- setup_ubuntu
    - ubuntuをいれ，その中にpythonを入れる方法
- setup_python
	- Docker Hubからpythonを引っ張る方法
	- こちらの方がスマート
	- debianがついてくるみたいでaptなど使える
